## About

This project has been created with _@adobe/create-ccweb-add-on_. As an example, this Add-on demonstrates how to get started with Add-on development using React and TypeScript with Document Sandbox Runtime.

## Tools

-   HTML
-   CSS
-   React
-   TypeScript

## Setup

1. To install the dependencies, run `npm install`.
2. To build the application, run `npm run build`.
3. To start the application, run `npm run start`.

## Running the Add-on in Adobe Express

1. Run `npm install` inside the `brandguard-express` folder.
2. Start the local dev server using `npm run start`.
3. Open https://new.express.adobe.com in your browser.
4. Open or create any design.
5. Open the Add-ons panel and load the local add-on (development mode).
6. Select **BrandGuard AI** and click **Analyze Design Structure**.

The add-on analyzes the structural complexity of the design
(layers, grouping, visibility, and locking) using the Document Sandbox runtime.
